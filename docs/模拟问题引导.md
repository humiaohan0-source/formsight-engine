# 模拟问题引导

这份模板用于 FormSight / 见相的用户材料收集。

它不是心理测评问卷，也不是算命表格。它的作用是帮助用户提供足够真实、具体、可参与模拟的人生材料，让模型能把经历、关系、资产、身体状态、行为模式和关键转折拆成变量，再生成多条可能人生线。

建议用户至少填写 1500 字以上。材料越具体，报告越像本人；材料太少，结果会变得泛化。

## 给用户的开场说明

```text
这个不是算命，也不是选择题测试。

FormSight/见相需要你提供比较完整的人生材料，然后模型会把这些材料拆成变量，模拟出几条可能的人生线。

你提供得越具体，报告越像你；如果材料太少，结果就会很泛。

下面这些问题不用写得很文学，真实、具体、直接就行。能写多少写多少，但建议至少 1500 字以上。
```

## 1. 基础信息

```text
1. 年龄：
2. 性别：
3. 所在城市 / 常住地：
4. 当前身份：学生 / 工作 / 自由职业 / 创业 / 待业 / 其他
5. 当前最想看清的问题是什么？
   例如：事业、关系、家庭、钱、人生方向、是否转型、是否离开某个环境。
```

## 2. 当前人生处境

```text
请描述你现在最真实的生活状态。

可以包括：
- 你现在每天主要在做什么？
- 你卡住的地方是什么？
- 你最焦虑的事情是什么？
- 你最想改变但一直没改变的事情是什么？
- 你现在最不敢面对的现实是什么？
- 如果未来一年什么都不变，你最害怕会发生什么？
```

## 3. 最近一年重要事件

```text
请列出最近一年对你影响最大的 5-10 件事。

每件事尽量写：
- 发生了什么？
- 它对你造成了什么影响？
- 它改变了你对自己、关系、钱、未来的什么看法？

例如：
1. 换工作 / 离职 / 升职 / 失业
2. 分手 / 进入关系 / 结婚 / 亲密关系冲突
3. 家庭矛盾 / 家庭责任变化
4. 身体问题 / 睡眠问题 / 情绪波动
5. 负债 / 收入变化 / 大额支出
6. 搬家 / 城市变化 / 圈层变化
7. 考试 / 毕业 / 学业问题
8. 创业 / 副业 / 内容创作 / 项目失败或启动
```

## 3A. 人生事件沉积补充

```text
请按年龄阶段补充一些“当时看起来不一定很大，但后来反复影响你”的事件。

1. 0-6 岁：家里、照料者、居住环境里，有没有让你特别害怕、安心、羞耻、被保护或被忽视的记忆？
2. 小学、初中、高中、大学/职校/研究生：每个阶段你和同学、老师、排名、评价系统的关系是什么样？
3. 有没有某件“小事”改变了你对自己的判断？
   例如：被笑、被夸、被孤立、被比较、被保护、被误解、突然被看见。
4. 这些事件发生后，有没有人帮你修复？
   例如：有人解释、有人站在你这边、后来关系变好、靠自己走出来、一直没有被处理。
5. 这件事后来在哪些选择里反复出现？
   例如：不敢暴露自己、不敢谈钱、总想证明、害怕冲突、容易突然退出、很难信任亲密关系。
6. 如果你觉得某段经历“不算大事但一直影响我”，也请写出来。模型会把它当成沉积变量，而不是普通背景信息。
```

## 4. 反复出现的行为模式

```text
请写出你身上反复出现的模式。

可以从这些角度写：
- 你遇到压力时通常会怎么反应？
- 你会逃避什么？
- 你会过度控制什么？
- 你是不是经常想很多但行动很少？
- 你是不是容易讨好别人？
- 你是不是经常拖延到最后一刻？
- 你是不是很难表达真实需求？
- 你是不是容易突然冲动做决定？
- 你是不是总在同一种关系/工作/选择里重复受伤？

请尽量给真实例子，不要只写形容词。
```

## 5. 家庭系统

```text
请描述你的家庭对你的影响。

可以包括：
- 你和父母/家人的关系怎样？
- 家庭对你有什么期待？
- 家庭是否在经济上支持你或控制你？
- 你最怕让家人失望的是什么？
- 家庭里有没有长期存在但没人明说的规则？
- 你做重大选择时，家庭压力会怎么影响你？
- 你希望和家庭的关系变成什么样？
```

## 5A. 求学经历、同学关系与代际家庭

```text
如果你愿意，请补充这些成长变量。它们会影响一个人后来如何做选择、如何面对评价、如何进入关系和新圈层。

1. 小学、初中、高中、大学/职校/研究生阶段，你分别是什么状态？
   例如：成绩、班级位置、是否被喜欢、是否被孤立、是否被老师认可、是否有长期压力。
2. 你和同学/同龄人的关系更像哪一种？
   融入、讨好、竞争、被排挤、旁观、隐藏自己、很会社交、只和少数人亲近。
3. 有没有某个老师、同学、考试、专业选择或校园事件，明显改变了你对自己的判断？
4. 你的父母/长辈身上有没有重复出现的人生命运？
   例如：为了家庭牺牲、一直忍耐、对钱恐惧、控制孩子、婚姻不幸福、事业受挫、很重面子。
5. 你最害怕自己重复家里谁的人生？又最想继承家里哪一种能力？
```

## 6. 亲密关系与朋友

```text
请描述你当前和过去重要关系里的模式。

可以包括：
- 你现在是否在亲密关系中？
- 这段关系给你支持多，还是消耗多？
- 你在关系里最常出现的问题是什么？
- 你会不会隐藏真实想法？
- 你会不会害怕冲突？
- 朋友通常怎么评价你？
- 你身边有没有能说真话的人？
- 你当前最缺什么样的人际支持？
```

## 7. 钱、工作和资产状态

```text
请描述你的现实资源和约束。

可以包括：
- 当前收入来源是什么？
- 收入是否稳定？
- 每月大概收入/支出区间？
- 是否有负债？大概多少？
- 存款能支撑几个月？
- 你有哪些技能资产？
- 你有哪些作品、经验、人脉、声誉资产？
- 钱对你来说更像安全感、自由、羞耻、压力，还是证明？
- 你现在做选择时，钱是怎么限制你的？
```

## 8. 身体、精力和心理状态

```text
请描述你的身体和精力状态。

可以包括：
- 睡眠怎么样？
- 身体有没有长期问题？
- 注意力是否容易集中？
- 最近一年压力对身体有什么影响？
- 你什么时候最有能量？
- 你什么时候最容易崩？
- 你有没有明显的情绪低谷、焦虑、麻木或失控时刻？
- 你的身体有没有在提醒你某种生活方式已经撑不住？
```

## 8A. 城市、阶层、身体形象、作息与信息摄入

```text
这些不是背景信息，而是会直接改变未来线的隐藏变量。

1. 你现在所在的城市/居住环境适合你吗？你有没有想过搬走、回老家、去大城市、换圈层？
2. 你有没有“自己和这个地方不匹配”的感觉？具体体现在哪里？
3. 钱对你来说有没有羞耻感、面子感、被看低感或阶层压力？
4. 你会不会用消费、存钱、赚钱、证明自己来处理不安全感？
5. 你对自己的外貌、身材、年龄、性别角色有什么长期压力？
6. 你的作息、睡眠、拖延、注意力稳定吗？什么时候最容易掉线？
7. 你平时最常刷什么内容？这些内容让你更清醒、更焦虑，还是更逃避？
8. 你最近反复收藏、搜索、转发但一直没有行动的内容是什么？
9. 你觉得人生最需要有意义的部分是什么？你有没有某种信念、直觉、命运感或精神支点？
```

## 8B. 世界观、行动感与未来感

```text
这部分会影响你后续怎么判断风险、怎么面对失败、怎么做长期选择。

1. 你觉得这个世界总体更像什么？
   例如：危险的、残酷竞争的、可以合作的、需要防备的、努力会有回报的、很多事靠运气的。
2. 你更偏唯心还是唯物？
   例如：你更相信现实证据、行动结果、钱和时间；还是更相信直觉、命运感、预兆、吸引力、精神意义？
3. 你做重大选择时，更常用什么判断？
   现实数据、家人意见、身体感觉、直觉、过往经验、别人成功案例、网络信息、命运感。
4. 你有没有一种感觉：努力也没用 / 做了也改变不了 / 只能等机会？
5. 你有没有一种能力：先试一小步，用结果验证，再决定要不要加码？
6. 你对未来的时间感是长还是短？
   能看到 3 年后的自己，还是只能撑过眼前几天/几个月？
7. 你现在有没有明显的年龄焦虑、来不及感、错过感？
8. 你最容易被过去哪件事固定住？它怎样影响你现在的判断？
```

## 9. 价值观和真正想要的未来

```text
请写你真正想要的未来，不要写“应该正确”的答案。

可以包括：
- 你最想过上一种什么样的生活？
- 你最想成为一个什么样的人？
- 你最不能接受自己变成什么样？
- 你愿意为了什么吃苦？
- 你不愿意再为了什么忍耐？
- 你羡慕什么样的人？
- 如果不考虑外界评价，你最想做什么？
```

## 10. 害怕的未来

```text
请写你最害怕自己走向哪种人生线。

可以包括：
- 你最怕 3 年后的自己变成什么样？
- 你最怕哪件事一直拖下去？
- 你最怕重复谁的人生？
- 你最怕在哪个关系/职业/城市/家庭结构里被困住？
- 你最怕别人怎么看你？
- 你最怕自己证明了什么不好的结论？
```

## 11. 关键转折点

```text
请回顾你人生里几个重要转折点。

建议写 3-8 个：
- 某次选择
- 某段关系
- 某次失败
- 某个身体事件
- 某次搬迁
- 某次考试 / 工作 / 创业节点
- 某个让你性格改变的经历

每个转折点写：
1. 当时发生了什么？
2. 你做了什么选择？
3. 如果当时走另一条路，人生可能会怎样？
```

## 12. 感情 / 婚恋补充问题
```text
如果你最想看感情、婚姻、伴侣选择，请尽量补充下面这些信息。
不需要写得很完美，但越具体，感情线越稳定。

1. 你现在的感情状态：
   单身 / 暧昧 / 恋爱 / 已婚 / 离异 / 关系不确定。

2. 你最想知道的问题是什么？
   例如：我会和什么样的人结婚？现在这个人适不适合我？婚后会不会变得很痛苦？我会不会被别人吸引？

3. 你过去最重要的 1-3 段关系：
   对方是什么样的人？你为什么被吸引？最后为什么继续或结束？
   这段关系后来有没有改变你的择偶标准、信任方式、事业选择、自我价值感？

4. 你在关系里最容易反复出现的模式：
   例如忍耐、冷战、讨好、控制、逃避、上头、反复失望、很难表达需求。

5. 你理想伴侣的具体样子：
   不要只写“对我好”，请写性格、工作/钱、家庭边界、情绪沟通、生活节奏、亲密方式。

6. 你绝对不能接受的伴侣类型：
   例如冷暴力、没责任感、妈宝、控制欲强、没有上进心、金钱观混乱、情绪不稳定。

7. 如果你已经有伴侣：
   你们最常吵什么？最满意哪里？最不满意哪里？你有没有想过分开？为什么还没分开？

8. 关于“被别人吸引”：
   你过去或现在有没有被另一种人吸引过？那个人给你的感觉是什么？他/她满足了你现有关系里缺少的哪一部分？

9. 关于婚姻现实：
   你对结婚、房子、钱、父母、彩礼/嫁妆、生育、城市选择、工作牺牲有什么担心？

10. 请提供一段真实互动：
   可以是一段聊天、一次吵架、一次失望、一次心动、一次你觉得“这个人可能不适合我”的瞬间。
```

## 13. 想让模型重点模拟什么

```text
最后，请选 1-3 个你最想看的方向：

A. 事业 / 工作 / 创业
B. 亲密关系 / 婚恋
C. 家庭关系
D. 钱和资产
E. 身体和精力
F. 城市 / 环境 / 圈层
G. 自我价值感
H. 未来 1-3 年的关键选择

也可以自己写。
```

## 14. 可选补充材料

```text
如果你愿意，也可以补充：

- 朋友圈 / 小红书 / 微博 / 公众号里你写过的自我表达
- 你最近反复收藏或转发的内容
- 你和朋友 / 伴侣 / 家人的典型聊天片段
- 你写过的日记
- 你做过的作品或项目链接
- 你过往做选择时最典型的一次纠结过程
```

## 收尾说明

```text
你不用把自己写得很完美。

这个模型最需要的不是“正确答案”，而是真实材料：
你的矛盾、拖延、反复、害怕、想要、资源和限制。

材料越真实，模拟出来的三条人生线越有参考价值。
```
