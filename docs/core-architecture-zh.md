# 核心架构

FormSight / 见相 的核心架构是：

```text
种子材料
-> 人生图谱
-> 内在 agent + 外部 agent
-> 人生状态仿真
-> 耦合与校准
-> 平行世界采样
-> 命运档案报告
-> 可视化仿真板
```

英文表达：

```text
Seed Material
-> Life Graph
-> Inner & External Agents
-> Life-State Simulation
-> Coupling & Calibration
-> Parallel World Sampling
-> Fate Archive Report
-> Visual Simulation Board
```

## 1. 种子材料

用户或示例案例提供的原始人生材料。

可以包括：

- 当前问题
- 人生经历
- 家庭关系
- 社会角色
- 资产状态
- 城市 / 行业环境
- 关系网络
- 健康能量
- 朋友观察
- 日记 / 简历 / 自我描述

开源样例必须使用虚构或充分匿名化材料。

## 2. 人生图谱

把原始材料整理成结构化图谱。

图谱里包含：

- 人
- 事件
- 关系
- 欲望
- 恐惧
- 资源
- 约束
- 反复模式
- 关键记忆
- 外部压力

## 3. Agent 层

把图谱里的力量变成 agent。

内在 agent 示例：

- 证明者
- 控制者
- 逃离者
- 讨好者
- 审判者
- 观察者

外部 agent 示例：

- 家庭期待
- 行业市场
- 城市压力
- 支持网络
- 能量系统
- 资产约束

## 4. 人生状态仿真

让个人世界往前运行。

每轮模拟：

```text
当前人生状态
-> 外部事件发生
-> 内在 agent 反应
-> 外部 agent 施压或支持
-> agent 记忆更新
-> 状态变量变化
-> 分支压力变化
-> 进入下一轮
```

## 5. 耦合与校准

FormSight 的关键不只是识别单个标签，而是识别变量之间的耦合。

示例：

```text
家庭期待 x 低机动性 -> 转型延迟
证明者 x 高可见度 x 低恢复 -> 机会与耗竭同增
逃离者 x 高压力 x 低承接 -> 突然切断
支持网络 x 外部观察 -> 干预线增强
```

校准层会解释：

- 哪些判断置信度高
- 哪些判断只是弱信号
- 哪些材料维度还不够
- 哪些矛盾信号需要保留

## 6. 平行世界采样

生成多个未来分支。

当前基础分支：

- 默认人生线：当前模式继续
- 隐秘转向线：关键变量先动起来
- 断裂重启线：压抑太久后突然重排

后续可以扩展：

- 事业优先线
- 关系优先线
- 稳定线
- 冒险线
- 整合线

## 7. 命运档案报告

把模拟结果翻译成用户能理解、愿意讨论、愿意分享、愿意复现的报告形态。

报告应该展示：

- 当前问题
- 材料质量
- 世界状态
- agent 力量
- 耦合命中
- 预测置信度
- 多轮模拟过程
- 三条人生线
- 不确定性来源

## 8. 可视化仿真板

可视化不是装饰，而是帮助用户看见模拟如何运行。

当前可视化包括：

- simulation pipeline
- agent field
- coupling network
- branch pressure
- branch evolution timeline
- agent memory
- parallel life scenes

这让 FormSight 更接近一个可检查的模拟系统，而不是一篇普通 AI 报告。
