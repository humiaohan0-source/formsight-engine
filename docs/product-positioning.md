# Product Positioning

## Core Claim

FormSight / 见相 is not a tool that knows the future.

It is an open-source engine that simulates possible futures from a person's repeated patterns, reality variables, inner agents, external agents, and coupling rules.

## User-Facing Framing

Use:

- parallel life simulation
- cognitive fate archive
- inner agent world
- life trajectory simulation
- observer-calibrated self-model
- personal life-world simulation

Avoid:

- fortune telling
- guaranteed destiny prediction
- psychological diagnosis
- supernatural claim
- clinical treatment language

## GitHub Framing

For GitHub, the project should be framed as:

> FormSight: an experimental agent-based life simulation engine for autobiographical seed material.

This sounds more serious than:

> AI fortune-telling app.

## Why It Is More Than A Simple H5

The HTML report or H5 page is only a surface.

The engine is the actual core:

- structured seed material
- life graph
- inner agents
- external agents
- coupling rules
- prediction calibration
- agent memory
- branch pressure evolution
- visualization

## Current Open-Source Direction

The current direction is full free open source.

That means:

- publish the runnable engine
- publish the structured knowledge base
- publish sample seeds
- publish generated demo reports
- publish the visual simulation board
- invite feedback through GitHub

Commercial packaging is not the current priority.

## Contribution-Oriented Translation

The same engine can support:

- CLI demo
- local HTML report
- local HTML/SVG visualizer
- fictional case studies
- research-style experiments
- future optional web frontend

The important thing is not to make the project look like a paid test. The important thing is to make the mechanism visible, inspectable, and discussable.
